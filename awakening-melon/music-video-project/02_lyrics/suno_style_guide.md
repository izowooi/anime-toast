# Suno AI 프롬프트 가이드 - 짧은 버전

## Style 필드에 입력할 내용

```
J-Pop, Female Vocal, Clear Voice, Piano, Acoustic Guitar, Strings Section, Dynamic, Building Energy, Bright Atmosphere
```

## Exclude Styles (제외할 요소)

```
Male Vocal, Dark Atmosphere, Aggressive, Heavy Distortion, Fast Tempo
```

## 추천 파라미터 설정

- **Vocal Gender**: Female
- **Style Influence**: 70-80% (스타일을 명확하게 반영)
- **Weirdness**: 30-40% (안정적이면서도 약간의 창의성)

## 곡 구조 설명

### 1. Verse 1 (시작부)
- **특징**: 잔잔하고 맑은 느낌으로 시작
- **메타 태그**: `[Gentle]`, `[Slow Tempo]`, `[Female Vocal]`
- **악기**: Piano, Acoustic Guitar (minimal)

### 2. Verse 2 (전개)
- **특징**: 감정이 점차 쌓이기 시작
- **메타 태그**: `[Clear Voice]`, `[Peaceful]`
- **악기**: Piano + Strings 추가

### 3. Pre-Chorus (준비)
- **특징**: 에너지가 점점 올라감
- **메타 태그**: `[Building Energy]`, `[Medium Tempo]`
- **악기**: 드럼 추가, Strings 강화

### 4. Chorus (클라이맥스)
- **특징**: 감정의 정점, 파워풀하고 고조된 느낌
- **메타 태그**: `[High Energy]`, `[Powerful Vocals]`, `[Dynamic]`
- **악기**: Full band + Strings Section

### 5. Bridge (감정 승화)
- **특징**: 밝고 희망찬 분위기
- **메타 태그**: `[Euphoric]`, `[Bright Atmosphere]`
- **악기**: Full orchestration

### 6. Outro (마무리)
- **특징**: 다시 차분하게 돌아오는 엔딩
- **메타 태그**: `[Gentle]`, `[Slow Tempo]`, `[Calm]`
- **악기**: Piano + Acoustic Guitar (처음과 같은 분위기)

## 주요 특징

✅ **인트로 없음**: 바로 가사로 시작
✅ **또박또박 들리는 보컬**: Clear Voice, Articulated
✅ **맑은 느낌**: Bright Atmosphere, Clear Voice
✅ **잔잔한 시작**: Gentle, Slow Tempo
✅ **점진적 고조**: Building Energy → High Energy
✅ **차분한 마무리**: Gentle, Calm로 회귀

## Suno AI 입력 방법

1. **Lyrics 필드**: `lyrics_short_version.txt` 파일 내용 복사
2. **Style 필드**: 위의 "Style 필드에 입력할 내용" 복사
3. **Exclude Styles**: 위의 "Exclude Styles" 내용 입력
4. **파라미터 조정**: 추천 설정 적용
5. **Generate** 클릭!
